# Openourplat

A catalog of free IT courses. Browse, search, and track your learning progress.

## Features

- **Course catalog** — browse courses by language and level
- **Lesson viewer** — step-by-step lessons with syntax highlighting (Prism.js)
- **Progress tracking** — save courses, track in-progress lessons, mark as completed
- **Dashboard** — overview of saved, in-progress, and completed courses
- **Data export/import** — backup and restore your learning data

## Tech

- **Languages** — HTML, CSS, JavaScript (vanilla)
- **Font** — [Roboto](https://fonts.google.com/specimen/Roboto) (Google Fonts)
- **Icons** — [Font Awesome](https://fontawesome.com/) (free kit)
- **Syntax highlighting** — [Prism.js](https://prismjs.com/)
- **Data** — fetched from [FakeApi](https://github.com/vatislo/FakeApi/)
